SHOWRUNNER 실행 방법
====================

■ Mac 사용자
  "start-mac.command" 파일을 더블클릭하세요.

  만약 "확인되지 않은 개발자" 경고가 뜨면:
  1) 파일을 마우스 우클릭(또는 control+클릭) → "열기" 선택
  2) 경고창에서 다시 "열기" 클릭
  (한 번만 해주면 다음부터는 그냥 더블클릭으로 실행됩니다)

■ Windows 사용자
  "start-windows.bat" 파일을 더블클릭하세요.

  만약 "Windows의 PC 보호" 경고가 뜨면:
  1) "추가 정보" 클릭
  2) "실행" 버튼 클릭

■ 최초 실행 시 필요한 것
  - Node.js가 설치되어 있어야 합니다 (없으면 설치 안내 메시지가 뜹니다)
    → https://nodejs.org 에서 LTS 버전 다운로드 후 설치
  - 처음 실행할 때만 자동으로 필요한 파일을 내려받습니다 (1~2분 소요, 인터넷 필요)
  - 그 다음부터는 실행 파일을 더블클릭하면 몇 초 안에 바로 켜집니다

■ 종료 방법
  실행 중 열려 있는 검은 터미널 창을 닫으면 서버가 종료됩니다.

■ 문제가 있을 때
  터미널 창에 나온 오류 메시지를 확인해 주세요.
